第四章 / 网络运行安全 / 第二十一条
法规末级条款义务（仅叶子节点）
CSL.Art21
网络运营者应当按照网络安全等级保护制度的要求，履行安全保护义务。
组织需建立并持续执行与其系统等级相匹配的安全管理和技术措施，确保网络运行安全并防止数据泄露、篡改、丢失。
