SAMMY 风格交互原型

目标：任何页面都保持左侧菜单栏，右侧仅展示末级内容；删除工作流、子分类筛选等低价值块。

01 控制型框架布局
适用于 ASVS / NIST / ISO / CIS / SCF / CCM 等控制标准。
02 成熟度框架布局
适用于 SAMM / BSIMM / DSOMM / AIMA 等成熟度模型。
03 法规型框架布局
适用于 中国法规 / 欧盟法规 / NIS2 / DORA 等义务型条款。
04 映射工作台布局
左侧源控制项，右侧映射目标控制项。去掉顶部大面积统计卡。
05 框架-布局矩阵
基于当前仓库框架清单，给出推荐布局模式与原因。
这批是静态 HTML 原型，仅用于确认信息架构与交互密度，不影响现有业务页面。
